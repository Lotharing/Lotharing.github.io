#《教程》
MySQL数据库教程

这篇文章主要讲解了，购买腾讯云MySQL服务器之后，怎么配置以及怎么使用。

购买腾讯云MySQL数据库

  • purchase

配置MySQL

  • 运行MySQL实例

  • 设置MySQL的账户和密码，一般账户是root，密码不要太过于简单还有一些字符编码和网络等属性。

  • 实例右侧管理，开启外网地址( cdb-**.bj.tencentcdb.com:10022)，内网本身就有(172.21.0.5:3306 )

  • 实例右侧登录，用root账户和密码登录进去数据库，有个可视化界面，可以进行建立数据库和建立表操作

对腾讯云MySQL的连接以及使用

  • 通过CMD管理员身份登陆，输入 mysql -h xxxxx.com -p 端口号 -u 用户名 -p 密码(控制台相关操作)

  • 通过内网地址连接数据库，msyql -h 172.21.0.5 -p 3306 -u root -p password

  • 通过使用MySQLWorkBench建立新连接，连接地址是cdb-**.bj.tencentcdb.com，端口号是10022，账户root，密码 ***

本地可视化工具连接

  • 如往常一样进行，建库，建表等操作。

项目的引用

  • 在配置文件中，建立连接： jdbc:mysql://cdb-**.bj.tencentcdb.com:10022/sys?useSSL=true”

注意事项

原因是 MySQL 5.5.45+, 5.6.26+ and 5.7.6这些版本的数据库需要手动指定SSL是否开启，所以原来的连接字符串：
jdbc.url=jdbc:mysql://127.0.0.1:3306/db_name?userUnicode=true&characterEncoding=UTF8就不可以了。
需要在其后附加useSSL=true或false，使用新的连接字符串：
jdbc.url=jdbc:mysql://127.0.0.1:3306/db_name?userUnicode=true&characterEncoding=UTF8&useSSL=false问题就解决了。
