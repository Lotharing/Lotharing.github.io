阿里云服务器的搭建

教程记录了阿里云从购买到搭建完成使用的过程

Posted by LuTong.Zhao on 2018-09-03
#《教程》
搭建个人服务教程

这篇文章将详细介绍，阿里云服务器 购买->配置->远程连接->部署项目

购买

  • 进入阿里云官网

  • 点击右上角登录按钮(使用的是淘宝账号密码,没有就去注册个淘宝账号吧)

  • 点头像后点击实名认证,点击支付宝认证

  • 立即认证后进入支付宝登录界面,输入支付宝验证码进行验证登录

  • 点击学生认证完善全部信息完成学生认证（这也是获得学生优惠特权的必要条件）可在支付宝App中填写，也可以直接在阿里云账号管理->学生认证中填写

  • 下载阿里云App,并安装,使用app登录阿里云客户端(淘宝账号密码)

  • 学生专区->学生特权->云计算特权->云服务器点击购买

  • 本人选择的是windows Server 2008 标准版 SP2 32位

  • 你会发现原先价格124/月的，给学生是9.87/月,不得不说很人性，我喜欢

远程主机配置服务器

  • 在网页版阿里云页面 产品与服务->云服务器ECS->实例 你可以看到已经购买到的服务器

  • 把公网IP地址记录起来，等下要用

  • 实例最右边 更多选项->连接管理终端->修改管理终端密码（该密码是连接的验证）

  • 返回 更多选项->重置密码（该密码是远程主机的登录密码）

  • 完成上述步骤 勾选实例-点击重启

本地电脑连接服务器

  • 打开你的控制台，选择—–>本实例安全组

  • 选择————->配置规则

  • 配置一个规则，因为安全组规定你的服务器开放那些端口号，而远程连接的端口号则是3389配置完成后第一步完成。

  • 因为你的服务器要被外网访问所以你需要开放外网端口，而系统默认的是80端口，故你还需要配置一个安全组规则（入方向）

  • DOS命令 mstsc (即mircrosoft terminal service client：微软终端服务客户端)

  • 输入公网Ip地址，用户名Administrator

  • 文件共享（本地资源->详细信息->选择驱动器->勾选你要共享的硬盘）

  • 连接->输入密码（自己看提示是哪个,笔者忘了）->进入远程主机页面

  • 配置完成后你需要在你的服务器上开放一下IP4的权限

  • .计算机 右键 管理：配置->入站规则->（文件打印机之类自己需要的规则点开启用）

部署项目（WEB为例）

  • 解压一份与计算机位数相同的TOMCAT服务器

  • WEB项目打WAR包放进webapps目录下

  • 启动TOMCAT（它可以自己解析该项目）

  • 访问默认8080端口，当然你现在用的是服务器，你可以用其他电脑根据服务器公网IP进行访问

  • 例如 我部署的SSMS 密码肯定不能告诉你！

  • 但你会发现我没有指定8080端口就可以访问

  • 把TOMCAT目录下的server.xml打开

Connector port=”8080” protocol=”HTTP/1.1” connectionTimeout=”20000” redirectPort=”8443” />

  • 把port改为80端口，因为默认就是访问80端口，所以你会看见我的访问连接中没有写端口号

  • 外网访问的功能

Host name=”localhost” appBase=”webapps” unpackWARs=”true” autoDeploy=”true”>

  • localhost 改为自己的外网IP地址

  • 访问项目路径就是–> http://外网IP/项目名称

  • 要记得每次做修改Tomcat需要重启才能生效

  • 域名指定（HOST标签中加入）

  • 域名解析设置为公网IP
